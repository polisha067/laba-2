## подробное описание через работу файлов и функций

   - файл const.py
    - хранит постоянные значения для работы программы
      - "LOG_FILE = "sell.log"" - имя файла для логгирования
      - "FORBID_PATHS" - запрещенные пути для удаления
      - "FILE_ENCOD = "urf-8"" - кодировка для работы с файлами 

   - файл loggg.py
      - обеспечивает запись всех команд и ошибок в файл sell.log
      - три функции для логгирования:
         - zapisat_log() - для успешных команд
         - zapisat_fold() - для ошибочных 
         - zapisat_predosterezhenie() - для предупреждений

   - файл sell.log
      - основной цикл программы
      - работа:
         - приветсвует)
         - запускает бесконечный цикл
            - входной показ
            - читает команду
            - проверка существования 
            - разбирает команду и аргумент
            - выполняет команду
            - обработка резултата и ошибки

## разбор команд

   - ls
   показывает список файлов и папок
      - "os.listdir()" - получение списка файлов
      - "os.stat()" - информация о файле
      - "pathlib.Path" - работа с путями

   - cd
   меняет текущую рабочую директорию 
      - "os.chdir()" - смена директории
      - "os.path.expanduser()" - раскрытие ~
      - "pathlib.Path" - проверка пути

   - cat
   выводит содержимое файла
      - "pathlib.Path" - работа с путями
      - "open()" - чтение файла
      - Обработка кодировок

   - cp
   копирует файл или директорию
      - "shutil.copy2()" - копирование файла 
      - "shutil.copytree()" - рекурсивное копирование директории
      - "pathlib.Path" - работа с путями

   - mv
   перемещает, переименовывает файл, директорию
      - "shutil.move()" - перемещение/переименование
      - "pathlib.Path" - работа с путями

   - rm
   удаляет файл,директорию
      - "os.remove()" - удаление файла
      - "shutil.rmtree()" - рекурсивное удаление директории
      - "pathlib.Path" - работа с путями
